# Текущий Контроль: Приложение «Площади фигур» (Вариант 14)

## Автор проекта
* **Студент:** Абазов Байсалбек
* **Вариант:** №14
* **Стек:** C#, WPF, .NET Framework, MSTest

## Описание задания
Разработка программного обеспечения для вычисления площадей геометрических фигур с динамическим пользовательским интерфейсом. Основные функции:
* Расчет площади прямоугольника по двум сторонам.
* Расчет площади круга по радиусу.
* Расчет площади треугольника по трем сторонам (формула Герона).

## Особенности реализации
* **Динамический UI:** Поля ввода и подписи (Label) отображаются только для тех параметров, которые необходимы выбранной фигуре.
* **Валидация:** Реализована проверка входных данных на корректность (запрет ввода текста, отрицательных чисел и проверка существования треугольника).
* **Документирование:** Код содержит XAML-комментарии для разметки и подробные комментарии к логике на C#.

## Инструкция по запуску
1. Склонируйте репозиторий на локальную машину.
2. Откройте файл решения `.sln` в среде Visual Studio.
3. Выполните сборку проекта (Build Solution).
4. Запустите тесты через «Обозреватель тестов» (Test Explorer), чтобы подтвердить работоспособность.
